Live Streaming

In an effort to help dispel the myth that many security people are able to break things without trying or thinking, I decided to start live-streaming my desktop while doing various security-related things. The aim is to show that:

  1. I am stupid. I don’t know what I am doing! It’s ok if you don’t either!
  2. Security stuff is tricky, finnicky, and frustrating. It’s also awesome fun, when it all finally comes together.
  3. Attacking binaries, web applications, or infrastructure from an offensive security perspective is an ongoing process of exploration and repeated failure. It doesn’t just magically happen.
  4. We can all learn something from each other, especially when we’re failing!

I will also spend some time on development as well, probably with a focus on (but not limited to) Metasploit and Meterpreter.

Hopefully you get a kick out of watching. Thanks for supporting the idea.

Links

  • Live streams appear on my Twitch.tv channel.
  • All the videos that are streamed are archived on my YouTube channel and on my Vimeo channel. YouTube can be picky with copyright content, and hence archiving in both locations makes sense!
  • I’m archiving the binaries and exploits in a Github repository.
  • My partner in crime, JustinSteven also streams on his Twitch.tv channel, and is well worth watching.
  • I use VIM, a lot. The configuration I’m using can be found on Github.
  • My GDB is set up to make use of gdb-peda.
  • While reverse engineering, I make use of Binary Ninja and IDA Pro.
  • I am a huge fan and frequent user of tmux.
  • My window manager is i3wm.
  • I run Fedora Core and Windows inside VMWare Workstation when doing streaming, but my host OS is Windows 10 Ultimate.

Suggestions

Do I take suggestions for streams? Absolutely! Hit me up with whatever it is you’d like to see, and I will do my best to accommodate you.

Rules

If you’re going to come along and watch the streams while they’re being recorded, please note the following rules:

  • Don’t be a dick to people. It won’t be tolerated. Multiple moderators are present in the chat during the streams, and you’ll be banned without a second chance if you turn nasty, troll, or don’t show respect to fellow viewers.
  • I have no problem with swearing (I do a bit myself!), but don’t direct it at people. Use it in jest or frustration, but keep it sane.
  • Please ask questions as we go. It’s easier to answer them in context than to do it all at the end. I periodically check the chat to see what questions are coming up, so don’t be shy.
