Gadget Me Up

Published 2006-11-08 by OJ Reeves

Read time: 1 min

I don’t know if you guys are free software gadget monkeys or not, but this stuff might be of value to those of you who use Visual Studio 2005. The guys over at SlickEdit.com, makers of the top notch SlickEdit editor, have released a stack of free gadgets for Visual Studio 2005 which are free! (I love that word.. I’m going to say it again.. FREEEEE).

The freebies include:

  • Editor Gadgets (Line ruler, Indentation guide, Auto-copy selection, Editor Graphic)
  • The Command Spy
  • File Explorer
  • Data Object Analyser
  • The SLOC Report
It’s a nice collection of tools if you ask me! Well worth playing with, so feel free to head over there and get your copy.
